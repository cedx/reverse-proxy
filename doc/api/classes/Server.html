close

(
  • [callback]
)
async

Stops the server from accepting new connections. The server is finally closed when all connections are ended and the server emits a 'close' event. Optionally, you can pass a callback to listen for the 'close' event.

Parameters:

  • [callback] Function optional

    A callback to invoke when server is finally closed.
